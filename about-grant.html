About the grant

In 2018 we have received the EU funding so we could implement the fastest, GPU-accelerated formula engine written in JavaScript.

Learn more
  • Name of the programme: Smart Growth Operational Programme 2014-2020 of the European Regional Development Fund under the Measure 1.1. R&D activity, Sub-measure 1.1.1 "Industrial research and development - work carried out by enterprises", co-funded by The National Centre for Research and Development.
  • Project name: "Development of the high-performance calculation engine for processing tabular data of the significant size on mobile devices and workstations using parallel computing and GPU."
  • Project number: POIR.01.01.01-00-0223/18
  • Beneficiary: Handsoncode sp. z o.o.
  • Total budget: PLN 1 121 375.00
  • Amount of grant: PLN 774 742.00
  • Duration: 2018-2019
  • Main goal: This strategic programme comprises actions targeted at creating the most efficient way to compute large tables containing Excel-like formulas using only JavaScript.
  • Projected results: We aim to create an efficient calculation engine to be released as a) an independent, open source, GPLv3-licensed library and b) a plugin to Handsontable Commercial Edition.
